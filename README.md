gmail osint tool made by Axom! 

this tool is made to track the gmail and see if it uses any of the websites listed in the modules. 
use able but not complete - under development

usage :
rm -rf ~/gmail-osint
git clone https://github.com/axomtools/gmail-osint.git
cd gmail-osint
python3 core.py user@gmail.com

- How does it work?
the tool tries to sign up/register the target gmail on the listed websites using the api, if it Already exists then the gmail does use this website.
